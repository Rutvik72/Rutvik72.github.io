    Facial Recognition

    Technology

    Accomplishments:

    Computer Vision was something I wanted to work on for a long time. So I figured the best way to get started was to create a program that recognizes faces. Upon research, many forums recommended OpenCV (developed by Intel) as the best library to create face detection program. However, OpenCV is not the only library required, Pillow (Python Imaging Library, also known as PIL), NumPy (support for multi-dimentional arrays and matrices) and Pickle module (serializing and de-serializing) are needed. Latter, I used basic machine learning - which I am still exploring - algorithms to train faces. In a subsidiary folder, I added images of various people including actors and myself, so that the algorithm extracts those images as numpy array with set coordinates and stores them for training - using .train() function. However, my prediction methods were fairly weak in recognizing the correct face. This project can be accessed through my GitHub.

    Next Step:

    I will further explore accurate face recognition algorithms in my next project where I use Nikon D3400, a DSLR camera, to retrieve live feed into Ubuntu (open source software operating system) supported by gphoto2 library and conduct face recognition as well as eye and smile detection. The purpose of this project is to become proficient in python and learn new method of designing acurate predictions. My inspiration to work on this project came from L3 WESCAM - they specialize in gyro-stabilized and EO-IR imaging systems for military.

    deltaDraw

    Major League Hackathon

    Accomplishments:

  • Designed a website which can import any .jpg image
  • Set up a server (integration of Flask and ngrok) for images to be uploaded though the deltaDraw website and later exported to a host computer
  • Convert images to etch sketch layout as well as extracting coordinates using image processing libraries
  • Mechanizing motors which holds a marker through Arduino after image has been processed
  • All done within the time span of 24 hours

    • Description

    Ever wonder what it would be like to have new drawings and/or artworks on your canvas every morning? Well, what if I told you our machine has the capability to draw any “.jpg” image you can import from google. One can import images through our website hosted on a local server. This tunneling system of transferring images from one computer to a host computer which processes the image is created using Flask and ngrok. Once the image the user wants to draw has been processed – converting it to etch sketch layout and extracting all of the coordinates – a file containing the coordinates is passed onto the Arduino. Next, the Arduino creates a set of instruction for the step motors, each mounted on the top two corners of the whiteboard. And voila, an etch sketch image is printed.
